Cover Page

This is a document produced by Bronagh Moynihan based on pages 1-22 of the IBM document Deployment and Conﬁguration Guide (available online here).

Conditions were applied to the text using a DITAVAL document. I devised a conditional processing scheme based on the following attributes:

  • Windows
  • Linux
  • System Administrator (Sys_Admin)
  • End User (End_User

Windows and Linux content was excluded/included as it pertained to the user. Technically advanced content was conditioned for System Administrators while content deemed basic was conditioned for End Users. The last topic in full is conditioned for system administrator. System admin content is flagged in red and yellow.
